# Xentu Creator

A cross-browser IDE for creating games with the [Xentu](https://github.com/xentu/xentu-engine) game engine.

![Screenshot of the editor in Windows 11](https://github.com/xentu/xentu-creator/blob/main/screenshot.jpg?raw=true)

## Features

- Written in C# .Net 6.0
- Uses the cross-platform Avalonia UI framework.
- Support for internationalisation (languages other than English coming soon).
- Rich text editor, with complex text selection, line-numbers, search, and more.
- Customisable syntax highlighting.
- Code completion and contextual insights for all the Xentu API.
- Low memory footprint (on average uses about 60-70mb).
- Game debugging, with a built-in console output window.

## How To Install

Click on the releases on this GitHub page, and download the version that suits
your operating system. Right now there is a Windows installer, and builds for
MacOS and Linux that are just extract and run affairs.

In the near future we'll publish an AppImage for MacOS and a Flatpak for Linux
to make the process more stream-lined.
